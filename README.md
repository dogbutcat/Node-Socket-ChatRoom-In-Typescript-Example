# Node-Socket-ChatRoom-In-Typescript

## Installation

1. install Dependences:
    ```sh
    npm install
    ```
2. add Typescript link:
    ```sh
    npm link typescript
    ```
    or run
    ```sh
    npm install typescript
    ```
3. use [Visual Studio Code](https://code.visualstudio.com) or run :
    ```sh
    npm start
    ```
4. Viste `http://localhost:8000`
